AngularJS 2 with Optimizely REST API

This AngularJS 2 app displays information about AB tests running in Optimizely using Optimizely REST API. This app doesn't offer any interaction because it was only designed to display LIVE results on a Television.

App in Action! Github project Technical details

This app is built using two Components namely AppComponent & ExperimentComponent. ExperimentComponent is a child component of AppComponent and it gets data from its parent using @Input() variable. This Component is responsible for displaying the data for an individual experiment.
'ClientServiceRemote' service class is responsible for retrieving data from Optimizely REST API. It uses AngularJS 2 http module to perform 'GET' requests which returns Observables to which we subscribe in the AppComponent class.

Panoramic virtual tours

These panoramic tours are created using Autopano and Panotour Pro software. The images were taken using Canon 5D camera and then stitched together using Autopano.
The locations are WAAPA (Western Australian Academy of Performing Arts), ECU ML Library (Edith Cowan University, Mount Lawley Campus Library) and ECU JO Library (Edith Cowan University, Joondalup Campus Library)
